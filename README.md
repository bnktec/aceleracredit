# Acelera Crédito — Apresentação Web

Apresentação estratégica da estruturação, posicionamento e plataforma tecnológica da **Acelera Crédito**.

## 🔗 Publicação

A página é servida via **GitHub Pages** a partir do arquivo [`index.html`](index.html):

**https://bnktec.github.io/aceleracredit/**

O arquivo é autossuficiente: a logo está incorporada (não exige pasta de imagens) e a fonte usa Google Fonts quando há internet, com fallback local.

## 🎛️ Controles

- **Setas**, **Page Up/Page Down** ou **barra de espaço** para navegar.
- Botão **Apresentar** para tela cheia.
